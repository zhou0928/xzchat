### AI问答
```
/ask ask 如何优化React性能？
/ask history
/ask search React
/ask popular
```

### 代码审查
```
/code-review create app.js "function test() { return 1; }"
/code-review report 123
/code-review list
```

### 文档摘要
```
/summarize create README.md "内容..." standard
/summarize templates
/summarize list
```

### 代码解释
```
/explain explain "function test() { return 1; }" --level=beginner
/explain list
/explain get 123
```

```
/docker list

/docker run myapp nginx
```

```

/metrics stats response_time 24h
/metrics get
```
